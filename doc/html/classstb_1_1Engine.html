Detailed Description

Main class for graphic conception.

This the base class of the project. You must inherit from this class to start using the graphic engine. Start by implementing the update and draw routine, related to the behaviour you expect in your game. In your main function, simply instantiate your child Engine, then call the mainLoop method to start the program.
